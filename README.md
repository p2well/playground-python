# playground-python
Python Playground

## Create Virtual Environment
```Python
python -m venv .venv
```

## Virtual Environment Activation

### Windows
```Python
.venv\Scripts\activate
```

### Linux
```Python
source .venv/bin/activate
```

## Install Requirements
```Python
pip install -r requirements.txt
```

## Upgrade pip
```Python
python -m pip install --upgrade pip
```
